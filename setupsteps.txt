1. helm install my-release oci://registry-1.docker.io/bitnamicharts/kafka -f values.yaml
2. Run kafaka-producer-application
3. Deploy kafka-start-event.bpmn
3. Trigger the produce at http://localhost:8081/publish/name where {name} is a string, and the content of the Kafka message.
4. For load testing, use the jmx file provided in Jmeter
